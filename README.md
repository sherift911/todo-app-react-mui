# 📝 Smart Todo App

A modern and responsive Todo application built with React and Material UI. The app allows users to efficiently manage their daily tasks with a clean and simple user interface.

## 🚀 Features

- Add new tasks
- Edit existing tasks
- Delete tasks
- Mark tasks as completed / not completed
- Filter tasks (All / Completed / Not Completed)
- Persistent data using Local Storage
- Fully responsive design for all screen sizes

## 🛠️ Tech Stack

- React JS
- Material UI (MUI)
- JavaScript (ES6+)
- CSS3
- Local Storage API

## 📱 Responsive Design

The application is fully responsive and optimized for:
- Mobile phones
- Tablets
- Desktop screens

## 📦 Installation

```bash
npm install
npm start
```
